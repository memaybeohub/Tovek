From bytecode soup to source you can read.

Most decompilers stop at correct. Tovek keeps going — recovering the names, services and idioms the Luau compiler stripped away, so the output reads like the original.

local function u1(p)
	for _, v7 in ipairs(p:GetChildren()) do
		if v7:IsA("BasePart") then
			local v8 = v7:Clone()
			v8.Parent = workspace
			v6.spawned = v6.spawned + 1
		end
	end
end

local v9 = v2.Heartbeat:Connect(function(p2)
	v6.elapsed = v6.elapsed + p2
	if v5.Health > 0 then
		print(("alive: %* · %*s"):format(v6.spawned, v6.elapsed))
	end
end)

// what makes it different

Engineered for readability.

A pipeline of analysis passes that the upstream decompiler never had — each one undoing another layer of what the compiler did to your code.

Real variable names

Locals and parameters are inferred from how they're used — never a wall of v1, v2, v3.

v7:Connect(…) → connection:Connect(…)

Handles preserved

GetService and require stay as one named header local instead of being inlined at every call site.

game:GetService("Players") ×N → declared once

OOP method recovery

Methods written with an explicit self are rendered back with colon-call syntax.

function T.f(self, x) → function T:f(x)

Reverses the -O2 optimizer

Single-use temporaries, inlined expressions and exploded table literals are reassembled — with the original inline points left as comments.

Idiomatic syntax

Compound assignments, backtick interpolation, normalized conditions and far fewer redundant parentheses.

x = x + 1 → x += 1

Deterministic & validated

Byte-identical output regardless of thread count, and the full corpus re-parses cleanly under Luau's own front end.

Batch tooling

decompile-folder and validate-folder chew through an entire SynSaveInstance dump in parallel.

Run it anywhere

Standalone CLI, a tiny HTTP server for the executor workflow, or a Cloudflare Worker for serverless deploys.

mimalloc fast path

The decompiler is allocation-bound; a per-thread allocator plus parallel per-function lifting deliver the speedups.

	table.insert(p._connections, p2)
	return p2
end
--  [-O2 INLINED, UNHOOKABLE] reconstructed definition;
local function cleanupConnections(p)
	for _, _connection in p._connections do
		_connection:Disconnect()
	end

	table.clear(p._connections)
end
